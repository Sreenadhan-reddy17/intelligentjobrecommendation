Under the Hood

Enterprise-Grade AI Architecture Powering SmartJob

The Hybrid Recommendation Pipeline

1. ATS Parsing Engine

Utilizes spaCy NLP and advanced RegEx to extract raw text, isolate structural blocks, and forcefully evaluate ATS compliance criteria. It instantly filters out non-resume documents via structural heuristics (Contact Info, Experience mapping).

2. Semantic Vectorization

Transforms parsed resumes into high-dimensional mathematical logic. A Sentence Transformer (all-MiniLM-L6-v2) encodes text into dense context embeddings, immediately establishing a semantic proximity matrix between candidates and roles.

3. Hybrid Blending

Fuses calculations mathematically: hybrid = α(Semantic) + β(Collab) + γ(Jaccard). It computes Cosine Similarity for semantics, strict Jaccard intersections for hard skills, and tracks multi-user matrix history for collaborative filtering.

Core Industry Engine Features

Cold-Start Auto-Adjustment

A common issue in matching engines is the "Cold Start" problem (no historical user interaction matrix). Our system mathematically detects empty interaction blocks and dynamically re-weighs the alpha and gamma coefficients toward structural Semantic Evaluation to ensure flawless accuracy on day zero.

Collaborative Intelligence (KNN)

As users interact with the platform, the backend records matrix behaviors. K-Nearest Neighbors (KNN) evaluates historical interactions to gently boost Job Role coefficients based on previous overlapping candidates.

Strict Applicant Evaluation

Prevents processing arbitrary inputs. Before vectors are even computed, the pipeline validates documents against a strict structural validation dictionary (Education headers, Contact identifiers) guaranteeing only true CVs make it into the funnel.

Jaccard Gap Analysis

Generates quantifiable missing skill subsets through Jaccard set operations. By explicitly identifying missing Set(A) - Set(B) components, candidates are given concrete career-roadmap tracking metrics pinpointing exact technical deficiencies.
